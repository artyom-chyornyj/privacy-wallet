# Privacy Wallet

A non-custodial wallet implementing Railgun privacy.

**TESTNET ONLY**

## Running Locally

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:3000`.

## Contributing

All contributions must pass linting and type checking before being merged.

```bash
npm run lint      # check for issues
npm run lint:fix  # auto-fix issues
npx tsc --noEmit  # type check
```

PPOI (Private Proof of Innocence) integration is required per the license and must not be removed or bypassed.
